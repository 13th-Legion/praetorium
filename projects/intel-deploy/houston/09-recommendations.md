# Recommendations

## Executive Summary

Based on the threat landscape analysis, the following recommendations are provided for conservative political organizations operating in the Houston metropolitan area.

---

## Security Recommendations

### Event Security

#### Pre-Event Planning

1. **Threat Assessment:** Conduct event-specific threat assessment 2-4 weeks prior
2. **Venue Selection:**
   - Avoid venues near synagogues or Jewish community centers (PSL targeting)
   - Prefer locations with controlled access and parking
   - Consider indoor venues to limit protest proximity
   - Verify venue security capabilities
3. **HPD Coordination:** Notify Houston Police Department for events over 100 attendees
4. **Intelligence Monitoring:** Monitor PSL Houston and Houston DSA social media for counter-protest announcements

#### Day-of Security

1. **Professional Security:** Engage licensed security personnel with crowd management experience
2. **Access Control:** Credential verification; bag checks; no-backpack policies
3. **Perimeter Security:** Establish buffer zones between event and public areas
4. **Medical Preparedness:** On-site medical personnel for events over 200 attendees
5. **Evacuation Planning:** Multiple exit routes; designated rally points

#### Vehicle Attack Mitigation

Given Bourbon Street attack methodology:
- Vehicle barriers at outdoor events
- Standoff distance from vehicle access points
- Awareness of rental truck threats

### Personnel Security (OPSEC)

#### Leadership Protection

1. **Personal Information:** Minimize public exposure of home addresses, family details
2. **Social Media:** Review and restrict personal account privacy settings
3. **Travel Security:** Vary routes; awareness of surveillance
4. **Residence Security:** Consider home security upgrades for high-profile individuals

#### Doxxing Prevention

1. **Domain Privacy:** WHOIS protection for organizational websites
2. **Corporate Records:** Use registered agent services
3. **Event Registration:** Avoid publishing attendee lists
4. **Staff Training:** Awareness of social engineering tactics

## Operational Recommendations

### Event Planning

#### Low-Risk Events (Internal meetings, small gatherings)

- Standard venue security
- Limited public announcement
- Basic access control

#### Medium-Risk Events (Public forums, candidate appearances)

- Professional security team (4-6 personnel)
- HPD notification
- Social media monitoring 1 week prior
- Credentialed access

#### High-Risk Events (Major rallies, controversial speakers)

- Enhanced security team (10+ personnel)
- HPD coordination and possible presence
- Counter-surveillance
- Legal observer/documentation
- Media management plan
- Medical team on-site
- Vehicle barriers (outdoor events)

### Incident Response

#### Protest Disruption

1. Do not engage with protesters
2. Document via video (discreetly)
3. Allow security to manage perimeter
4. Have spokesperson ready for media
5. Continue event if safely possible

#### Physical Confrontation

1. Immediate security response
2. Call 911
3. Evacuate if necessary
4. Preserve evidence
5. File police report
6. Engage legal counsel

#### Doxxing/Online Attack

1. Document all threats
2. Report to platforms
3. Report credible threats to FBI
4. Consider temporary social media blackout
5. Engage cybersecurity professionals if needed

## Legal Considerations

### Texas Protest Law

- Counter-protesters have First Amendment rights on public property
- Cannot block access to private events
- Open carry is legal; may be restricted at private venues
- Harassment and threats are criminal

### Documentation

- Video record all interactions with protesters
- Preserve social media posts/threats
- Maintain incident logs
- Photograph any damage

### Liability

- Verify venue liability coverage
- Consider event-specific insurance
- Have legal counsel on retainer

## Long-Term Recommendations

### Relationship Building

1. **Law Enforcement:** Develop relationships with HPD intelligence unit
2. **Venue Partners:** Establish preferred venue list with security-conscious locations
3. **Security Providers:** Pre-qualify security firms before needed
4. **Legal Support:** Retain counsel familiar with protest law and civil rights

### Organizational Preparedness

1. **Training:** Annual security awareness training for staff
2. **Protocols:** Written security protocols for different event types
3. **Communication:** Secure communication channels for sensitive planning
4. **Review:** Post-event security reviews and lessons learned

### Community Intelligence

1. **Monitor PSL Center:** Track opening and activities of PSL Houston's new organizing center
2. **Coalition Tracking:** Monitor for new left-wing coalition formations
3. **Incident Tracking:** Maintain database of local incidents for pattern analysis
