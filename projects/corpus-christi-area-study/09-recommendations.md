# Recommendations

## 9.1 Physical Security Measures

### Event Security

**Minimum Standards:**
- Professional security if needed
- Notify CCPD for events
- Controlled entry

**Enhanced for Higher Risk:**
- LE presence
- Perimeter management

## 9.2 Event Planning Considerations

**Pre-Event:**
- Monitor local social media
- Select low-profile venues

**During/Post:**
- Real-time awareness
- Debrief

## 9.3 OPSEC for Members

**Personal Guidance:**
- Privacy on social media
- Avoid identifiers at events

## 9.4 Monitoring & Early Warning

**Sources:**
- Local news, activist pages

**Cadence:**
- Weekly checks
- Pre-event intensification

## 9.5 Law Enforcement Coordination

- Build relationships with CCPD
- Advance notice of events

## 9.6 Legal Preparedness

- Retain counsel for civil rights/issues
- Document incidents

## 9.7 Summary: Priority Actions

### Immediate

1. Set up monitoring
2. Distribute OPSEC guide
3. Contact CCPD

### Near-Term

1. Develop event SOPs
2. Training for staff

### Ongoing

1. Maintain awareness
2. Annual review
